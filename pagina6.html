6. Como fazê-lo girar: se quiser, pode fazer com que o planador voe em diferentes direções. Faça dois pequenos cortes na parte posterior das asas para fazer ailerons, como no desenho. Para fazê-lo ir para a esquerda,

dobre o ailerom esquerdo para cima e o direito para baixo. Para fazê-lo ir para a direita, dobre o ailerom direito para cima e o esquerdo para abaixo.
